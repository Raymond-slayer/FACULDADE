# Voz Segura

## Sobre o projeto

O Voz Segura é uma aplicação web desenvolvida com o objetivo de auxiliar no enfrentamento da violência contra a mulher, oferecendo suporte por meio de informações, canais de denúncia, contatos de emergência e mecanismos de solicitação de ajuda de forma segura e acessível.

O projeto foi desenvolvido de forma simulada como estudo de caso da disciplina de Engenharia de Software, aplicando conceitos fundamentais da área como levantamento de requisitos, modelagem de sistemas, banco de dados e metodologias ágeis.

---

### Cliente

## Objetivo Geral

Desenvolver uma aplicação web capaz de:

- Disponibilizar informações sobre violência contra a mulher
- Permitir acesso a contatos de emergência
- Possibilitar envio de pedidos de ajuda
- Registrar denúncias
- Facilitar acesso à rede de apoio
- Garantir sigilo e segurança das informações

---

# Funcionalidades do sistema

## Usuária

### Cadastro
- Criar conta
- Informar dados pessoais

### Login
- Entrar no sistema por email e senha

### Contato de emergência
- Cadastrar contatos
- Salvar:
    - Nome
    - Email
    - Endereço
    - Telefone

### Denúncias
- Registrar ocorrência
- Informar:
    - Data
    - Tipo de violência
    - Detalhes
    - Nome do agressor
    - Grau de parentesco

### Pedido de ajuda

A usuária pode:

- Acionar contato de emergência
- Solicitar ajuda imediata
- Acionar apoio rápido

### Rede de apoio

Permite acesso a:

- Atendimento psicológico
- Informações
- Apoio especializado

### Relatórios

Visualização de:

- Pedidos de ajuda realizados
- Denúncias registradas
- Contatos acionados
- Histórico de atividades
